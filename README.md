#####Setup instruction step-by-step for node.js with express
1. brew install node
2. brew install npm
3. mkdir myExpress & cd myExpress
4. npm install express -g
5. express
6. npm install
7. npm start

Open browser and type this 'localhost:3000'.


#####Setup mongodb.
1. brew install mongodb
2. mkdir db
3. cd db
4. mongod --dbpath ./

You can see a mongodb's log.
